What makes a good UI? I like to explore the edges of how User Interface can work change experiences and while I like to explore out of the world 3D experiences created with Virtual Reality, I also like to find elegance in retro designs. So I wanted to create a design that resonates with retro + textureless 3d feel. As opposed to modern UI elements floating on top of our perceived background, neumorphic components pretend to extrude from the background. Check out my Open Source Code for Neumorphism in iOS.

The beauty of neumorphic surfaces in cards

As much fun it is to deal with the flat and minimal UI design patterns, there is an attraction in curved surfaces like imperfect paper. When put in a Swift UIView, and given positive and negative shawows with a slight contrast, there is a fresh feel to the whole experience. These cards do not flow like the modern cards, and instead feel like most of the card's back lies falt on surface with only slightly curved edges.

Neumorphic circular buttons

Concave and convex surfaces become more pronounced as the radius of the views changes. While rectangles with curved cirfaces look great as cards, circular surfaces serve as great buttons. With the appropriate images placed in the center of the buttons, the user experience can be greatly enhanced. One learning that I got from building the Favorite button was that it was paramount for such a to button have stark differences in images based on the states, otherwise it might be difficult to identify a state change and the current state of the button.


Challenges faced in user testing

  1. The adaptability of this trend is yet to be tested extensively. Feedback from my users so far has informned me of mixed reviews: with some users being really excited about this new UI, while others could not care less.
  2. The design seems technically feasible in web and iOS platforms, but I have to yet test it on Android and non-native plaforms./li>
  3. The lack of contrast flat out annoyed some of my users and this leads me to think that this might not be an accessible design. When doing accessibility test, this current design flat out failed and has to be changed drastically in order to be usable.

Takeaway

Neumorphism could be the future of UI design. Identifying up and coming patterns and the idea of working on something new brings me immense joy. As of January 2020, it seems fresh and new, it does come with potential caveats and has to be dealt with caution. Going forward, I would love to explore this pattern more by performing usability testing and user feedback. I would like to explore how the accessibility can be improved using different color and shading patterns and how I can improve the touch and feel of these componenets with the use of rich touches.
